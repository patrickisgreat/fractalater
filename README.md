## What is Fractalater?

Fractalater is a **high-performance fractal generator** that renders beautiful mathematical art directly in your browser using WebGL shaders. Create, explore, and save stunning fractals with real-time parameter adjustment.

- **Real-time rendering**: GPU-accelerated computation renders millions of pixels instantly
- **Interactive exploration**: Pan, zoom, and tweak parameters with immediate visual feedback
- **Save & share**: Store your favorite fractals and come back to continue exploring
- **Learn fractals**: Understand the math behind these mesmerizing patterns

## Features

| Feature | Description |
|---------|-------------|
| **WebGL Shaders** | GPU-powered rendering for smooth, real-time fractal generation |
| **Multiple Fractal Types** | Mandelbrot, Julia, Burning Ship, and Tricorn sets |
| **Color Schemes** | 6 built-in palettes: Classic, Smooth, Fire, Ocean, Rainbow, Monochrome |
| **Smooth Coloring** | Anti-aliased iteration count for beautiful gradient transitions |
| **User Accounts** | Register and login to save your fractal explorations |
| **Fractal Gallery** | Browse, load, and manage your saved fractals |
| **Public Sharing** | Make fractals public for others to discover |

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/yourusername/fractalater.git
cd fractalater
```

2. **Install dependencies**

```bash
npm install
```

3. **Set up the database**

```bash
npx prisma migrate dev
```

4. **Start the development server**

```bash
npm run dev
```

5. **Open your browser**

Navigate to [http://localhost:3000](http://localhost:3000) and start exploring fractals!

## Fractal Types

### Mandelbrot Set

The classic fractal. For each point `c` in the complex plane, iterate `z = z² + c` starting from `z = 0`. Points that don't escape to infinity are in the set.

### Julia Set

Related to the Mandelbrot set, but uses a fixed constant `c` while varying the starting point `z`. Different values of `c` produce wildly different patterns.

### Burning Ship

A variant that uses the absolute values of the real and imaginary parts: `z = (|Re(z)| + i|Im(z)|)² + c`. Creates flame-like structures.

### Tricorn

Also known as the Mandelbar set. Uses the complex conjugate: `z = z̄² + c`. Produces distinctive three-cornered shapes.

## Controls

### Navigation

| Action | Control |
|--------|---------|
| Pan | Click and drag |

| Zoom out | Scroll down |

### Parameters

| Parameter | Effect |
|-----------|--------|
| **Max Iterations** | Higher values reveal more detail at deep zoom levels |
| **Escape Radius** | The threshold for determining if a point escapes |
| **Julia Real/Imag** | The constant `c` for Julia sets (only shown for Julia type) |
| **Color Offset** | Shifts the color palette |
| **Color Scale** | Stretches or compresses the color mapping |

## Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) with App Router
- **Rendering**: WebGL with custom GLSL fragment shaders
- **Database**: SQLite with [Prisma 7](https://www.prisma.io/)
- **Authentication**: [NextAuth.js](https://next-auth.js.org/) (Auth.js)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Language**: TypeScript

## The Math Behind Fractals

Fractals are generated by iterating a simple formula and checking whether the result "escapes" to infinity or stays bounded. The color of each pixel represents how quickly (or if) that point escapes.

For the Mandelbrot set:

```
z₀ = 0
zₙ₊₁ = zₙ² + c

where c is the complex coordinate of the pixel
```

The magic comes from the boundary between escaping and non-escaping points, which reveals infinitely complex detail at every scale.
